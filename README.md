Visor Estelar – NASA APOD

Aplicación de escritorio desarrollada en Python que consume la API APOD de la NASA para visualizar imágenes astronómicas y su información asociada.

Descripción:
Este proyecto permite consultar imágenes espaciales proporcionadas por la NASA, retrocediendo por fechas hasta encontrar contenido válido. Incluye una interfaz gráfica de escritorio y manejo de datos desde una API externa.

Tecnologías utilizadas:
- Python
- PyQt5
- Requests
- NASA APOD API
- Googletrans

Funcionalidades:
- Consulta de imágenes astronómicas desde la NASA
- Navegación automática por fechas
- Visualización de imágenes y texto descriptivo
- Traducción automática del contenido
- Interfaz gráfica de escritorio

Estado del proyecto:
Proyecto académico y personal en desarrollo continuo.
